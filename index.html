Política de Privacidade - FaunaStats

Introdução

Esta política de privacidade descreve como o aplicativo FaunaStats coleta, usa e protege suas informações.

Informações Coletadas

O FaunaStats não coleta dados pessoais dos usuários. Todas as informações fornecidas pelo usuário permanecem localizadas no dispositivo e não são compartilhadas com terceiros.

Uso das Informações

Os dados inseridos no aplicativo são utilizados exclusivamente para os cálculos dos índices faunísticos e de diversidade. Não utilizamos esses dados para outros fins.

Segurança

Tomamos medidas para garantir que as informações permaneçam seguras no dispositivo do usuário. O aplicativo não faz upload de dados para a internet ou servidores remotos.

Alterações na Política de Privacidade

Podemos atualizar esta política de privacidade periodicamente. Qualquer alteração será comunicada aos usuários através desta página.
